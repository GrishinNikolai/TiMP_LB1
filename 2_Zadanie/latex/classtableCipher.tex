\doxysubsection{Подробное описание}
Класс для шифрования методом табличной маршрутной перестановки 

Реализует табличную маршрутную перестановку для русского алфавита. Маршрут записи\+: по горизонтали слева направо, сверху вниз. Маршрут считывания\+: сверху вниз, справа налево. Ключ -\/ количество столбцов таблицы.
