Подробное описание

Класс для шифрования методом Гронсфельда

Реализует шифр Гронсфельда для русского алфавита. Ключ устанавливается в конструкторе. Для зашифровывания и расшифровывания предназначены методы encrypt и decrypt.

Предупреждения
Реализация только для русского языка
